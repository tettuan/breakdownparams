# 用語集

このドキュメントは、breakdownparamsライブラリで使用される主要な用語の定義を提供します。

## 目的と意図

この用語集は以下の目的で作成されています：

1. **用語の統一**
   - ライブラリ内で使用される用語の定義を明確にし、一貫性を保つ
   - 開発者間でのコミュニケーションを円滑にする
   - ドキュメント全体での用語の使用を統一する

2. **理解の促進**
   - 各用語の背景や意図を理解しやすくする
   - 用語間の関連性を明確にする
   - 実装の意図を理解する助けとする

3. **参照の容易さ**
   - 関連する仕様書への直接リンクを提供
   - 用語の詳細な定義や使用例へのアクセスを容易にする
   - 実装の詳細を素早く確認できるようにする

## 基本用語

### パラメータ関連

| 用語       | 説明                                                                                                                                         | 参照仕様ファイル               |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------ |
| 位置引数   | コマンドラインでハイフンなしで指定される引数。ライブラリの主要な機能を制御するために使用され、引数の位置によって意味が決まる。               | [params.ja.md](params.ja.md)   |
| オプション | コマンドラインでハイフン付きで指定される引数（例：`--help`）。機能の詳細な制御や追加情報の指定に使用される。長形式と短縮形の両方をサポート。 | [options.ja.md](options.ja.md) |
| エイリアス | オプションの短縮形（例：`-h`は`--help`のエイリアス）。コマンドの簡潔な記述を可能にし、ユーザビリティを向上させる。                           | [options.ja.md](options.ja.md) |
| 長形式     | オプションの完全な形式（例：`--help`）。可読性と明確性を重視する場合に使用。ドキュメントやスクリプトでの使用を推奨。                         | [options.ja.md](options.ja.md) |
| 短縮形     | オプションの省略形式（例：`-h`）。コマンドラインでの迅速な入力のために使用。対話的な操作での使用を推奨。                                     | [options.ja.md](options.ja.md) |

### パラメータタイプ

| 用語         | 説明                                                                                                                    | 参照仕様ファイル             |
| ------------ | ----------------------------------------------------------------------------------------------------------------------- | ---------------------------- |
| NoParams     | 位置引数なしのパラメータタイプ。ヘルプ表示やバージョン確認など、単純な操作に使用。オプションのみの指定が可能。          | [params.ja.md](params.ja.md) |
| SingleParam  | 位置引数1つのパラメータタイプ。初期化などの単一の操作を実行するために使用。現在は`init`コマンドのみをサポート。         | [params.ja.md](params.ja.md) |
| DoubleParams | 位置引数2つのパラメータタイプ。メインの機能を実行するために使用。DemonstrativeTypeとLayerTypeの組み合わせで操作を指定。 | [params.ja.md](params.ja.md) |

### 主要な型

| 用語              | 説明                                                                                                                | 参照仕様ファイル                       |
| ----------------- | ------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| DemonstrativeType | 最初の位置引数の型（`to`, `summary`, `defect`）。実行する操作の種類を指定。各値は特定の処理を表し、型安全性を確保。 | [params.ja.md](params.ja.md)           |
| LayerType         | 2番目の位置引数の型（`project`, `issue`, `task`）。操作の対象となる層を指定。階層構造を表現し、処理の範囲を定義。   | [params.ja.md](params.ja.md)           |
| ParamsResult      | パラメータ解析結果の基本型。解析結果を型安全に扱うためのインターフェース。エラー情報も含む。                        | [development.ja.md](development.ja.md) |
| OptionParams      | オプションパラメータの型。オプションの値を型安全に扱うためのインターフェース。各オプションの値を保持。              | [development.ja.md](development.ja.md) |

## オプション関連

### 標準オプション

| オプション    | エイリアス | 説明                                                                                 | 参照仕様ファイル               |
| ------------- | ---------- | ------------------------------------------------------------------------------------ | ------------------------------ |
| --help        | -h         | ヘルプ情報を表示。ライブラリの使用方法や利用可能なオプションを確認するために使用。   | [options.ja.md](options.ja.md) |
| --version     | -v         | バージョン情報を表示。ライブラリの現在のバージョンを確認するために使用。             | [options.ja.md](options.ja.md) |
| --from        | -f         | ソースファイルパス。入力ファイルの場所を指定。相対パスと絶対パスの両方をサポート。   | [options.ja.md](options.ja.md) |
| --destination | -o         | 出力ファイルパス。結果の保存先を指定。パスの存在確認は行わない。                     | [options.ja.md](options.ja.md) |
| --input       | -i         | 入力レイヤータイプ。処理の入力元となる層を指定。LayerTypeの値のみを許可。            | [options.ja.md](options.ja.md) |
| --adaptation  | -a         | プロンプト適応タイプ。処理の挙動を調整するために使用。カスタマイズ可能な動作を指定。 | [options.ja.md](options.ja.md) |
| --config      | -c         | 設定ファイル名。DoubleParamsでのみ使用可能。処理の設定を外部ファイルから読み込む。   | [options.ja.md](options.ja.md) |

## 拡張機能関連

| 用語         | 説明                                                                                                               | 参照仕様ファイル                           |
| ------------ | ------------------------------------------------------------------------------------------------------------------ | ------------------------------------------ |
| 拡張モード   | カスタムなバリデーションルールを適用するモード。標準の制約を超えた柔軟な使用を可能にし、特定のユースケースに対応。 | [custom_params.ja.md](custom_params.ja.md) |
| 標準モード   | デフォルトのバリデーションルールを使用するモード。基本的な使用に適し、型安全性と一貫性を保証。                     | [custom_params.ja.md](custom_params.ja.md) |
| ParserConfig | パーサーの設定を定義するインターフェース。拡張モードでの動作を制御し、カスタムルールを指定するために使用。         | [custom_params.ja.md](custom_params.ja.md) |

## エラー関連

| 用語                 | 説明                                                                                             | 参照仕様ファイル                           |
| -------------------- | ------------------------------------------------------------------------------------------------ | ------------------------------------------ |
